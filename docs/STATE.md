## GEO product core concepts (inputs to discovery-researcher)
_Summary for the discovery-researcher to validate and expand:_
1. **GEO Flywheel (continuity logic):** Audit → Analyze → Strategize → Create → Distribute → Measure → Learn → loop. Daily monitoring, weekly sweep, monthly re-audit. Subscription stays because AI-search visibility decays without maintenance.
2. **Three-vector GEO Score (0–100):** BRAND (entity/authority presence: Wikidata, Crunchbase, LinkedIn, Google Business consistency, mention volume/sentiment, E-E-A-T) + PERFORMANCE (technical/measurable: schema.org, llms.txt, AI-crawler access, freshness, citation share-of-voice vs competitors, Google AI Overview source presence) + AI (the generation/intelligence engine across all stages).
3. **AI at every stage:** query simulation, gap analysis, content creation, mention classification, reporting.
4. **Capabilities in priority order (GEO Audit Engine = wedge/free-tier lead magnet):**
   - A. GEO Audit Engine — query ChatGPT/Claude/Perplexity/Gemini/Google AIO; detect/measure brand citations; baseline all 3 vectors.
   - B. Citation Monitor — weekly sweeps, mention rate/position/sentiment, competitor share-of-voice.
   - C. Strategy Generator — AI turns audit into prioritized dated plan + content calendar.
   - D. Multi-channel content engine — blog (owned, schema'd), LinkedIn, Reddit, Quora, FAQ.
   - E. Reddit module — monitor subreddits; AI drafts value-add answers; STRICTLY human-approved, ToS-safe, posted from client's own warmed account. NO autonomous posting.
   - F. Entity/Knowledge-Graph manager — Wikidata/Crunchbase/LinkedIn/Google Business consistency.
