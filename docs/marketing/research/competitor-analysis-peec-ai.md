# Competitor Analysis — Peec AI

> **TL;DR.** Peec AI (peec.ai) is an "AI search analytics for marketing teams" tool — pure **monitoring** of brand visibility across AI engines (ChatGPT, Perplexity, Gemini + AI Overviews/Copilot; more models in Enterprise). ICP = marketing teams & **agencies** (mid-market). Pricing ≈ **$95/mo** (brand entry) → **$245/mo** (agency); base plans include only **3 of 6 models** (extra models €30–140/mo each); unlimited seats; API/SSO Enterprise-only. Differentiators: **UI-scraping** methodology ("as users see it"), prompt management, source/citation discovery, multi-country, Looker/CSV/API/MCP exports, 2000+ teams + Lily Ray endorsement. **Their #1 admitted gap: monitoring-only — no execution** (no content creation / step-by-step fixes). That is exactly TrustIndex AI's wedge: **Audit + Execute** (Content Studio writes the fix; OrganicPosts does it for you), **all 5 engines included (no add-ons)**, and a **free→$29→$99** low-friction SMB funnel they don't serve. Action items A–D (prompt library, source-discovery panel, methodology transparency, export/API) below. Pricing figures vary by source/currency and change often — treat as approximate (captured 2026-06).

## 2. Where Peec is stronger (what to borrow)

1. **Prompt management as a first-class feature** — discover/organize/prioritize prompts by search volume ("Prompts are the foundation").
2. **UI-scraping methodology** ("as users see it") — high credibility vs. API-based reads.
3. **Source/citation discovery** — shows where mentions originate (G2, Reddit, LinkedIn, NYT).
4. **Multi-country tracking** per project.
5. **Data portability** — Looker Studio, CSV, API, MCP.
6. **Unlimited seats** on all plans (agency-friendly).
7. **Social proof + founder-led support** (direct Slack with founders).
8. **Clean, polished UI; fast onboarding.**

## 3. Where TrustIndex AI is (or can be) stronger — Peec's weaknesses

1. **Audit + EXECUTE.** Peec is monitoring-only — reviews note it "doesn't give step-by-step recommendations on what to create/update." TrustIndex writes the fix (**Content Studio**) and executes it (**OrganicPosts** DFY / GEO Sprint). **Primary wedge.**
2. **All 5 engines included, no add-ons** vs. Peec's 3-of-6 + €30–140/mo per model.
3. **Lower friction / down-market** — free 60s test → $29 Kit → $99 Growth. Peec starts ~$95/mo and targets agencies; TrustIndex owns the SMB self-serve long tail.
4. **Unified 3-vector TrustIndex Score (AI / Performance / Brand + overall)** — a simple, sellable narrative vs. loose metrics.
5. **Done-for-you arm (OrganicPosts)** — Peec has none. Service revenue + retention.
6. **Compliance posture** (LGPD+GDPR+CCPA artifacts, privacy-first). (Both lack SOC 2 — no decisive winner.)

## 4. What to use to improve the product (prioritized)

**Quick wins (high impact / low effort):**
- **A. Manageable prompt library** in the paid dashboard — show the tracked buyer prompts, allow add/edit, surface AI-suggested prompts. (Free test already exposes the single prompt; extend to a managed set in-product.)
- **B. "Sources the AI cites about you" panel** — package the Reddit deep-dive + entity-graph + citation data into a clean view of top cited domains/sources; feeds the content plan.
- **C. Methodology transparency** — keep showing the exact prompt (advantage); add an honest "measured via each model's API" note + put real-UI fidelity on the roadmap.
- **D. Data export + open API** — CSV export now (Looker/Sheets later); offer **API access NOT gated to Enterprise** (anti-lock-in wedge vs. Peec).

**Bigger bets:**
- **E. Multi-country / multi-market** tracking per brand (today US/EU only).
- **F. UI-fidelity scraping** (selective) — credibility play; weigh build cost + ToS risk vs. messaging.
- **G. Social proof** — currently none (pre-launch); execute the founder-as-case-study plan fast.

**Positioning copy to hammer:**
- "They monitor. We audit **and execute**."
- "All 5 AI engines included — no per-model add-ons."
- "Start free in 60s — no card." (vs. their ~$95/mo entry)
